## Week subtopics

- Function
  - Syntax
    - Reserved word 'function'
    - Function name
    - Function parameters
  - Invocation
    - Function arguments
  - Return statement
- Primity types methods
  - string
    - concat()
    - includes()
    - split()
    - charAt()
    - slice()
    - indexOf()
    - toUpperCase() / toLowerCase()
    - replace()
  - number
    - Math.floor()
    - Math.round()
    - Math.abs()
    - +, -, \*, /, %
  - boolean
    - NOT (!)
    - AND (&&)
    - OR (||)
  - Object
    - Literal object ({ })
    - Class Object (new Object)
    - Square brackets property access (obj['prop'])
    - Dot property access (obj.prop)
  - Object by reference or value?
