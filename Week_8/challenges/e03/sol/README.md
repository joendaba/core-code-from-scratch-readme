# Even or odd

## Solution 1 😎

```javascript
function evenOrOdd(number) {
  if (number % 2 === 0) {
    return "Even";
  }
  return "Odd";
}
```

## Solution 2 😎

```javascript
function evenOrOdd(number) {
  return number % 2 === 0 ? "Even" : "Odd";
}
```
